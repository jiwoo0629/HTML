o <font> : 행의 글자 크기, 색상, 글꼴 지정
 <font size = "n"> : 크기 (숫자가 작을수록 작음)
  n 자리에 +n / -n 넣을 경우 본문에 상대적인 크기 적용
 <font color = " "> </font> : 색상
 <font face = " "> </font> : 글꼴
 자동 줄바꿈 x -> <br>로 줄바꿈 해줘야함

 - <font> 태그 외에 다양한 효과들
 <b> ~ </b> / <strong> ~ </strong> : 진하게
 <i> ~ </i> / <em> ~ </em> : 이탤릭체
 <u> ~ </u> : 밑줄
 <sub> ~ </sub> : 아래 첨자
 <sup> ~ </sup> : 위 첨자
 <big> ~ </big> : 본문보다 2포인트 정도 크게
 <small> ~ </small> : 본문보다 2포인트 정도 작게
 <strike> ~ </strike> : 태그 내 글자에 삭제 표시
 <tt> ~ </tt> : 가로 세로 비율이 고정된 글꼴

 <blockquote> : 다른 글을 인용할 경우 사용
 태그 내의 내용이 다른 문장보다 약간 들여쓰여짐
 <address> : 주소/연락처를 기술할 때 사용
 이태릭체로 나타남

o 기초, 목록을 나타내는 태그들
 <li> : 실제 내용을 나타낼 때 사용
 <ul> : 순서가 없는 목록 (unordered list)
  type - disc : 검은 원, circle : 흰 원, square : 사각형
 <ol> : 순서가 있는 목록 (ordered list)
  type - 1 : 숫자, a : 영어 소문자, A : 영어 대문자
         i : 소문자 로마 숫자, I : 대문자 로마 숫자
         disc : 검은 원, circle : 흰 원, square : 사각형
  start - 숫자 : 시작 숫자/문자 지정
 <dl> : 정의 목록 나열 (definition list)
  <lh> ~ </lh> : 머리글 제목
  <dt> : 용어의 제목
  <dd> : 용어의 설명

o 움직이는 텍스트 <marquee>
 - align : 텍스트의 위치
  상 : top, 중 : middle, 하 : bottom
 - height, width : 텍스트가 나타낼 범위
 - hspace, vspace : 텍스트 바깥 쪽의 상하좌우 여백
 - behavior : 흘러갈 스타일
  scroll : default 값, 페이지를 가르며 계속 흘러감
  side : 한쪽 끝에 닿으면 정지
  alternate : 일정 범위 내에서 왔다 갔다 반복
 - direction : 흘러갈 방향
  위 : up, 아래 : down, 왼쪽 : left, 오른쪽 : right
 - scrolldelay : 스크롤 지연 속도(숫자가 클수록 더 느림)
 - scrollamount : 한번에 스크롤되는 픽셀 수
  (숫자가 클수록 더 빨라짐)
 - bgcolor : 문자열 배경색
 - loop : 스크롤 반복 횟수
